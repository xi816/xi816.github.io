There is some letters with the same sound. Here are rules for them:
  • ḣ before consonants, x before vowels.
  • ś’ before consonants, č’ before vowels.
  • ž before consonants, ğ before vowels.
  • ź before consonants, ǧ before vowels.
  • ẗ before consonants, tʎ before vowels.
If the letter is at the end of a word, then you check the first letter: obyx (x, because a word starts with a vowel). Graphemes that are written with these letters have only one correct writing: qxg /qχ/ is written with x in it, even though x should be written only before vowels.

Grammatical genders

Because this language should be the hardest, it has 6 grammatical genders (not really a lot, but with 25 genders it's already 500 person pronouns):
  1. Masculine;
  2. Feminine;
  3. Neuter;
  4. Table;
  5. Food;
  6. Dark matter.


Person and Number

There's 5 person forms and 4 number forms.
The person forms are:
  • 0th -- No one;
  • 1st -- I, We;
  • 2nd -- You;
  • 3rd -- He, She, It;
  • 4th -- Everyone.
The number forms are:
  • Singular;
  • Dual;
  • Trial;
  • Plural.

Nouns

Nouns should end with:
  • -al
  • -us
  • -u
  • -os
  • -um
  • -an
  • -a
The suffix doesn't change from gender, you should remember the correct one for every root.

Here are all noun cases: 
    
 Nominative | -     | N
 Accusative | -ta   | N~o
 Gentitive  | -trun | of N
 Dative     | -tsy  | to N
 Innesive   | -tia  | in/at N
 Adessive   | -tor  | on N
 Objective  | -t’as | about N
 Essive     | -tyny | from N
 Fassive    | -tuẙy | as N
Case suffix is written after the noun suffix: -antor, not -toran. If you want to write someting, that is a letter, you should separate it with dots. Example: fy is /ɸ/, but if you want to write /fɤ/, you would write f.y
